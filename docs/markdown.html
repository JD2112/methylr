10.1. How to convert .Rmd to .ipynb¶

  1. Require an R package, rmd2jupyter

devtools::install_github("mkearney/rmd2jupyter")
library(rmd2jupyter)

  1. Prepare the .Rmd file

  2. Convert the file

rmd2jupyter("sample_file.Rmd")
